Программа закрытой секции включает панельную дискуссию, посвященную вопросам обеспечения информационной безопасности финансовых организаций, анализу опыта проведения киберучений, а также проблеме качества аудита информационной безопасности с участием представителей Банка России. Во второй части секции эксперты поделятся практическими советами по формированию и реализации дорожной карты по ИБ-комплаенсу на 2021-2022 г.г. Подробная программа.

Участие в секции для финансовых организаций-участниц Сообщества АБИСС бесплатно. Подробнее о регистрации для участия в секции и вступлении в Сообщество можно узнать на сайте АБИСС.

Подзаголовок

  • Основой для создания платформы NBT является модульная платформа. NET Core c открытым исходным кодом и совместимая с операционными системами Windows, Linux, macOS
  • В основе архитектуры целостные функциональные блоки в виде отдельных модулей, c помощью которых происходит управление иерархией пакетов и их версионностью
  • Стандартные языки программирования и фреймворков ускоряет и упрощает разработку
